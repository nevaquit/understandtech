# v1.0.0 Tag Checklist — Core Platform

Use this list before `git tag -a v1.0.0`. **Stream upload, Stripe, Postmark, and payment E2E are post-v1.0.0** — see [phase-7-production.md](phase-7-production.md).

## Required gates

| # | Gate | How to verify |
|---|------|----------------|
| 1 | Production HTTPS | `curl -sI https://understandtech.app/` → 200 + `cf-ray` |
| 2 | AI Worker health | `curl -s https://ai.understandtech.app/health` → `{"status":"ok"}` |
| 3 | AI auth gate | `POST /tutor` without JWT → 401 |
| 4 | Origin Pulls | Direct VM IP HTTPS fails; traffic via Cloudflare only |
| 5 | Key Vault core secrets | Six OK: DB, Redis, Anthropic, OpenAI, worker shared (no `REPLACE-ME`) |
| 6 | Redis sessions | `\core\session\redis` on VM; `PONG` from Redis |
| 7 | Self-hosted runner | `gh api …/actions/runners` → `understandtech-web-prod` online |
| 8 | Playwright core | `cd tests/e2e && npx playwright test --project=chromium --workers=1` → 8/8 |
| 9 | Smoke script | `./scripts/smoke-test-deployment.sh` → 0 failures (optional rows may WARN) |
| 10 | Rollback doc | [playbook §7.4](playbook.md#74-rollback-plan) + [post-deployment-validation.md](post-deployment-validation.md) |
| 11 | Post-deploy checklist | Run [post-deployment-validation.md](post-deployment-validation.md) (skip deferred rows) |

## Tag and deploy

Full sequence: [v1.0.0-core-release.md](v1.0.0-core-release.md).

```bash
git tag -a v1.0.0 -m "Core production release — LMS, AI tutor, CI/CD (billing/Stream deferred)"
git push origin v1.0.0
gh workflow run deploy.yml --ref v1.0.0
gh run watch
PROD_URL=https://understandtech.app ./scripts/smoke-test-deployment.sh
```

Then complete [post-deployment-validation.md](post-deployment-validation.md) within 30 minutes.
