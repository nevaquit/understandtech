# History
Access past Agent conversations from the history panel.
## Opening history
- Click the history icon in Agent sidepane
- Press `Opt Cmd '`
## Managing chats
- \*\*Edit titles\*\*: Click to rename
- \*\*Delete\*\*: Remove unneeded chats
- \*\*Open\*\*: Click to review full conversation
