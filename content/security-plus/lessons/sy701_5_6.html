Exam objective 5.6

Given a scenario, implement security awareness practices

Security awareness practices are essential for educating employees and users about security risks and their role in protecting organizational assets. A strong security culture is a key defense mechanism.

Security Awareness

  • Phishing: Training users to identify and report phishing attempts.
    • Campaigns: Running simulated phishing campaigns to test user awareness and identify areas for improvement.
    • Recognizing a phishing attempt: Educating users on common phishing indicators (e.g., suspicious sender, generic greetings, urgent requests, bad grammar, malicious links).
    • Responding to reported suspicious messages: Establishing clear procedures for users to report suspicious emails and messages.
  • Anomalous behavior recognition: Training users to identify and report unusual or suspicious activities.
    • Insider threat: Educating employees about the risks of insider threats and how to report suspicious behavior.
  • User guidance and training:
    • Policy/handbooks: Providing clear and concise security policies and handbooks.
    • Situational awareness: Training users to be aware of their surroundings and potential security risks in various environments.
    • Password management: Educating users on creating strong, unique passwords and using password managers.
    • Removable media and cables: Training on the secure handling of USB drives, external hard drives, and network cables.
    • Social engineering: Educating users about various social engineering tactics (e.g., pretexting, impersonation, tailgating) and how to resist them.
    • Operational security (OpSec): Training on protecting sensitive information and operations from adversaries.
    • Hybrid/remote work: Providing specific security guidance for employees working remotely or in hybrid environments.
    • Risky environments: Educating users about increased risks in public Wi-Fi, shared computers, or unsecured locations.
  • Reporting and monitoring: Establishing clear channels for reporting security incidents and monitoring compliance with security policies.
    • Initial: Onboarding security awareness training for new employees.
    • Recurring: Regular, ongoing security awareness training to reinforce concepts and address new threats.
  • Development: Designing and developing effective security awareness programs.
  • Execution: Delivering security awareness training through various methods (e.g., online modules, in-person sessions, posters, newsletters).
