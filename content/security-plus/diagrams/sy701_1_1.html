🎯 Security Controls Classification Matrix

🔧 Technical Controls

Technology-implemented safeguards:

  • Firewalls: Filter network traffic based on rules
  • Antivirus/Anti-malware: Detect and remove malicious software
  • IDS/IPS: Monitor and prevent intrusions
  • Encryption: Protect data confidentiality and integrity
  • Multi-Factor Authentication: Additional identity verification
  • SIEM: Centralized security information management
  • DLP: Data loss prevention systems

📋 Managerial Controls

Administrative policies and procedures:

  • Security Policies: High-level security directives
  • Risk Assessments: Systematic threat identification
  • Training Programs: Security awareness and education
  • Incident Response Plans: Structured breach response
  • Business Continuity Plans: Operational resilience
  • Background Checks: Personnel security screening
  • Change Management: Controlled system modifications

👥 Operational Controls

Day-to-day human-implemented processes:

  • Security Guards: Physical premise monitoring
  • User Account Management: Access provisioning/deprovisioning
  • Security Audits: Regular compliance assessments
  • Backup Operations: Data protection procedures
  • Configuration Management: System baseline maintenance
  • Separation of Duties: Fraud prevention through role division
  • Vulnerability Management: Systematic weakness remediation

🏢 Physical Controls

Tangible security measures:

  • Access Control Systems: Card readers, biometric scanners
  • CCTV Systems: Visual surveillance and monitoring
  • Environmental Controls: HVAC, fire suppression, power
  • Perimeter Security: Fencing, barriers, lighting
  • Cable Management: Secure data transmission paths
  • Device Security: Laptop locks, equipment fastening
  • Secure Disposal: Media destruction and sanitization

🔍 Security Control Functions

🛡️ Preventive Controls
Stop incidents before they occur:
• Firewalls blocking malicious traffic
• Access controls preventing unauthorized entry
• Antivirus preventing malware execution
• Security awareness preventing social engineering
⚠️ Deterrent Controls
Discourage potential attackers:
• Visible security cameras
• Warning signs about monitoring
• Security guard presence
• Legal consequences documentation
🔍 Detective Controls
Identify ongoing or completed incidents:
• Intrusion detection systems
• SIEM correlation engines
• Security audit logs
• Motion detection sensors
🔧 Corrective Controls
Remediate after incident detection:
• Incident response procedures
• System backup restoration
• Patch management deployment
• Malware quarantine systems
⚖️ Compensating Controls
Alternative protection methods:
• Manual processes when automation fails
• Additional monitoring for encryption gaps
• Enhanced logging for access control limitations
• Segregation of duties for single-person tasks
📋 Directive Controls
Guide behavior through documentation:
• Security policies and procedures
• Technical implementation standards
• Operational guidelines
• Compliance requirements
