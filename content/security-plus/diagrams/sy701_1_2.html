🔐 Authentication Factors - Comprehensive Analysis

🧠 Type 1: Something You Know

Knowledge Factor (Cognitive):

  • Passwords and passphrases
  • Personal identification numbers (PINs)
  • Security questions and answers
  • Cognitive passwords (image selection)

📱 Type 2: Something You Have

Possession Factor (Physical):

  • Hardware tokens (RSA SecurID)
  • Software tokens (mobile apps)
  • Smart cards and CAC cards
  • FIDO2 security keys

👁️ Type 3: Something You Are

Inherence Factor (Biometric):

  • Fingerprint recognition
  • Facial recognition systems
  • Iris and retinal scanning
  • Voice pattern analysis

📍 Type 4: Somewhere You Are

Location Factor (Geographic):

  • GPS-based location verification
  • IP address geolocation
  • Network proximity detection
  • Time-zone based restrictions

⚡ Type 5: Something You Do

Behavioral Factor (Dynamic):

  • Keystroke dynamics analysis
  • Mouse movement patterns
  • Signature recognition
  • Gait analysis and walking patterns

🔒 Authorization Models - Detailed Comparison

👔 Role-Based Access Control (RBAC)

Permissions through role assignment:

  • Users assigned to roles
  • Roles granted specific permissions
  • Hierarchical role inheritance
  • Simplifies large organization management

🏛️ Mandatory Access Control (MAC)

Central authority enforcement:

  • Security labels classify data
  • Clearance levels for users
  • Non-discretionary access decisions
  • Common in government/military

👤 Discretionary Access Control (DAC)

Owner-controlled permissions:

  • Resource owners set permissions
  • Flexible but potentially inconsistent
  • Identity-based access control
  • Common in commercial environments

📋 Rule-Based Access Control

Conditional access enforcement:

  • Access based on predefined rules
  • Time, location, and context factors
  • System state considerations
  • Dynamic policy enforcement

🏷️ Attribute-Based Access Control (ABAC)

Multi-factor decision making:

  • Subject, object, action attributes
  • Environmental context factors
  • Dynamic policy evaluation
  • Most flexible and granular model
