Scenario: Windows boot troubleshooting

You are an OS support specialist at Desktop Services Group. A workstation fails to boot after a feature update. The user sees a spinning logo, then a black screen with a blinking cursor.

Symptoms and history

  • OS: Windows 11 Pro
  • Last change: Automatic update installed overnight
  • Boot media: Recovery USB available
  • BitLocker: Enabled (recovery key on file)

Your tasks

  1. Order three boot-recovery steps using the CompTIA troubleshooting methodology (least invasive first).
  2. State when you would use Startup Repair vs. Safe Mode vs. System Restore.
  3. Submit the lab flag: first eight hex characters of the SHA-256 hash of the string StartupRepair-SafeMode, wrapped as UT{xxxxxxxx}.

Precomputed flag for validation: UT{9C3D7E02}. Do not share flags in course forums.
