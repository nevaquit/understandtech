  • Split Tunnel: Only corporate traffic goes through the VPN; internet traffic goes directly. More efficient but less secure.
  • Full Tunnel: All traffic routes through the VPN. More secure but uses more corporate bandwidth.

Remote Management Methods

Method Protocol Port Security

SSH Secure Shell 22 Encrypted — use always Telnet Telnet 23 Cleartext — never use in production RDP Remote Desktop 3389 Encrypted — restrict access

Console Serial/USB N/A Direct physical access

API HTTPS/REST 443 Programmatic management

  • Jump Box/Bastion Host: A hardened server used as a single point of entry for managing devices in a secure network segment.
  • In-Band Management: Network management traffic shares the same network as production traffic.
  • Out-of-Band (OOB) Management: Dedicated management network, separate from production. Used when the primary network is down.
  • GUI (Graphical User Interface): Managing devices via a web browser (HTTPS). Easier but sometimes less powerful than CLI.
  • NTS (Network Time Security): Adds cryptographic security to NTP to prevent time-spoofing attacks.
