Exam tip

If two routes have the same AD, the router uses the metric to choose. If both AD and metric are equal, the router will load-balance across both paths. Address Translation • NAT (Network Address Translation): Translates private IP addresses to a public IP address. • PAT (Port Address Translation): Maps multiple private IPs to a single public IP using different port numbers. Also called "NAT Overload." This is what your home router does. First Hop Redundancy Protocol (FHRP) Provides a virtual default gateway shared between multiple routers, ensuring clients can still reach the internet if one router fails. • HSRP (Hot Standby Router Protocol): Cisco proprietary. • VRRP (Virtual Router Redundancy Protocol): Open standard.
