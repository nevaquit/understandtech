Exam focus

Summarize various types of attacks and their impact to the network

  • Ransomware: Encrypts data and demands payment for the decryption key.
  • Trojan: Malicious software disguised as legitimate software.
  • Worm: Malware that spreads by exploiting vulnerabilities without user interaction.
  • Spyware: Secretly monitors user activity and steals data.
  • VLAN Hopping via Double Tagging: An attacker sends a frame with two 802.1Q tags. The switch strips the outer tag (native VLAN) and forwards the frame to the VLAN in the inner tag. Mitigation: Never use VLAN 1 as the native VLAN; explicitly tag all ports.
