Exam focus

Given a scenario, use appropriate IPv4 network addressing

  • Loopback/Localhost: 127.0.0.1 — tests the local network stack.
  • APIPA: 169.254.x.x — automatically assigned when DHCP fails.
  • If a device has a 169.254.x.x address, it cannot reach the DHCP server. This is the first thing to check when a user reports "no internet connection." Verify the DHCP server is running and the physical connection is good. Subnetting Quick Reference (CIDR Cheat Sheet) CIDR Subnet Mask Network Bits Host Bits Total Addresses Usable Hosts /24 255.255.255.0 24 8 256 254 /25 255.255.255.128 25 7 128 126 /26 255.255.255.192 26 6 64 62 /27 255.255.255.224 27 5 32 30 /28 255.255.255.240 28 4 16 14 /29 255.255.255.248 29 3 8 6 /30 255.255.255.252 30 2 4 2 Formula: Usable Hosts = 2^(host bits) - 2
  • The Magic Number Method: Subtract the last non-zero octet of the subnet mask from 256 to find the block size. For /26 (255.255.255.192): 256 - 192 = 64. Subnets start at 0, 64, 128, 192. If a host has IP 192.168.1.75 with /26, it is in the 192.168.1.64 subnet (range .64-.127).
