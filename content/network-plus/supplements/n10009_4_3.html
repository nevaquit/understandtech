Exam focus

Given a scenario, apply network security features, defense techniques, and solutions

  • Disable all unused ports and services.
  • Change all default usernames and passwords immediately.
  • Enable encrypted management protocols (SSH over Telnet, HTTPS over HTTP).
  • Apply the principle of least privilege to all accounts.
  • Keep firmware and software patched and up to date.
  • Enable logging and send logs to a centralized SIEM.
  • Port Security: Limits the number of MAC addresses on a switch port; can shut down the port if violated.
  • 802.1X: Port-based authentication using a RADIUS server. Involves a supplicant (client), authenticator (switch/AP), and authentication server (RADIUS).
  • MAC Filtering: Allows or denies access based on a device's MAC address. Easily bypassed by MAC spoofing.
  • Standard ACL: Filters based on source IP address only.
  • Extended ACL: Filters based on source/destination IP, protocol, and port number.
  • Implicit Deny: All ACLs end with an implicit deny — if traffic is not explicitly permitted, it is blocked.
  • URL Filtering: Blocking access to specific websites based on their domain name.
  • Content Filtering: Inspecting the actual content of a web page or email and blocking it if it contains inappropriate or malicious material.
  • DMZ (Demilitarized Zone): Place public-facing servers (web, email, DNS) in the DMZ. The DMZ is accessible from the internet but separated from the internal network by a firewall. If a DMZ server is compromised, the attacker cannot directly reach internal resources.
