Exam focus

Explain concepts related to the Open Systems Interconnection (OSI) reference model

  • Memory Aid: "Please Do Not Throw Sausage Pizza Away" (Physical → Application)
  • You WILL be asked to identify which OSI layer a protocol, device, or problem belongs to. Switches = Layer 2. Routers = Layer 3. Firewalls can operate at Layers 3, 4, or 7. Hubs = Layer 1.
  • TCP is connection-oriented (3-way handshake: SYN, SYN-ACK, ACK). UDP is connectionless and faster but unreliable. VoIP and streaming use UDP. File transfers use TCP.
