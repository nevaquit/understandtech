802.11 Wireless Standards

Standard Frequency Max Speed Key Feature

802.11a 5 GHz 54 Mbps First 5 GHz standard 802.11b 2.4 GHz 11 Mbps First widely adopted standard 802.11g 2.4 GHz 54 Mbps Backward compatible with 802.11b 802.11n (Wi-Fi 4) 2.4 / 5 GHz 600 Mbps MIMO introduced 802.11ac (Wi-Fi 5) 5 GHz 3.5 Gbps MU-MIMO; beamforming 802.11ax (Wi-Fi 6) 2.4 / 5 / 6 GHz 9.6 Gbps OFDMA; improved dense environments

Exam tip

The 2.4 GHz band has only 3 non-overlapping channels (1, 6, 11). The 5 GHz band has many more non-overlapping channels, making it less susceptible to interference. Wi-Fi 6 (802.11ax) adds the 6 GHz band. Wireless Encryption Protocol Security Level Notes WEP Broken Never use — easily cracked in minutes WPA Weak Uses TKIP — deprecated WPA2 Strong Uses AES/CCMP — current standard WPA3 Strongest SAE replaces PSK; provides forward secrecy Wireless Authentication • PSK (Pre-Shared Key): A single password shared by all users. Suitable for home and small office. • Enterprise (802.1X): Uses a RADIUS server for individual user authentication. Required for corporate environments. AP Deployment • Autonomous AP: Self-contained; configured individually. Good for small deployments. • Lightweight AP (LWAP): Managed centrally by a Wireless LAN Controller (WLC). Required for large enterprise deployments.
