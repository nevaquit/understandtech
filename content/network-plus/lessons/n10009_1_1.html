Exam objective N10-009 1.1

Explain concepts related to the Open Systems Interconnection (OSI) reference model

This lesson aligns with the CompTIA Network+ N10-009 exam blueprint (Version 4.0). Focus on how the concept appears in enterprise and cloud networking scenarios.

Key topics

  • Layer 1 Physical — bits on the wire; cables, hubs, NICs
  • Layer 2 Data link — frames, MAC addresses, switches
  • Layer 3 Network — packets, IP addressing, routers
  • Layer 4 Transport — TCP vs UDP, ports, reliability
  • Layer 5 Session — dialog control between applications
  • Layer 6 Presentation — encryption, compression, encoding
  • Layer 7 Application — HTTP, DNS, SMTP and user-facing protocols

Study approach

  • Relate the objective to OSI layers, defense-in-depth, and documentation where applicable.
  • Practice explaining trade-offs (performance vs security, cost, and availability).
