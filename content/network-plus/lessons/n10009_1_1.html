Exam objective N10-009 1.1

Explain concepts related to the Open Systems Interconnection (OSI) reference model

The OSI model is a seven-layer conceptual framework that standardizes network communication. Every protocol, device, and troubleshooting step maps to one of these layers.

Layer#NamePDUKey protocols / devices
Application7ApplicationDataHTTP, HTTPS, FTP, DNS, SMTP, SNMP, SSH
Presentation6PresentationDataSSL/TLS, JPEG, ASCII, MPEG
Session5SessionDataNetBIOS, PPTP, RPC
Transport4TransportSegment/DatagramTCP, UDP
Network3NetworkPacketIPv4, IPv6, ICMP, IPsec, Routers
Data Link2Data LinkFrameEthernet, MAC, Switches, Bridges
Physical1PhysicalBitCables, Hubs, Repeaters, NICs

Memory Aid: "Please Do Not Throw Sausage Pizza Away" (Physical → Application)

Exam tip

You WILL be asked to identify which OSI layer a protocol, device, or problem belongs to. Switches = Layer 2. Routers = Layer 3. Firewalls can operate at Layers 3, 4, or 7. Hubs = Layer 1.

Exam tip

TCP is connection-oriented (3-way handshake: SYN, SYN-ACK, ACK). UDP is connectionless and faster but unreliable. VoIP and streaming use UDP. File transfers use TCP.
