Exam objective N10-009 4.1

Explain the importance of basic network security concepts

Common Security Terminology

  • Risk: The likelihood that a threat will exploit a vulnerability.
  • Vulnerability: A weakness in a system or process.
  • Exploit: A tool or technique used to take advantage of a vulnerability.
  • Threat: Any potential danger to an asset.

The CIA Triad

Principle Definition Implementation

Confidentiality Data is only accessible to authorized parties

Encryption, access controls, MFA

Integrity Data is not altered or tampered with Hashing (MD5, SHA), digital signatures Availability Data and services are accessible when needed

Redundancy, backups, DDoS protection

Identity and Access Management (IAM)

Technology Function

MFA Requires two or more verification factors (something you know, have, or are) SSO (Single Sign-On) One set of credentials grants access to multiple applications RADIUS Centralized AAA (Authentication, Authorization, Accounting) for network access LDAP Protocol for accessing and maintaining directory services (e.g., Active Directory) SAML XML-based standard for exchanging authentication data between identity provider and service provider TACACS+ Cisco's AAA protocol; separates authentication, authorization, and accounting; encrypts entire payload

Pro tip

RADIUS vs. TACACS+: RADIUS is an open standard and encrypts only the password. TACACS+ is Cisco proprietary and encrypts the entire payload. TACACS+ is preferred for device administration (routers, switches). RADIUS is preferred for network access (VPN, 802.1X). Access Control Concepts • Role-Based Access Control (RBAC): Assigning permissions based on a user's job function rather than individually. • Least Privilege: Giving users only the minimum permissions necessary to do their job. • Time-based Authentication: Restricting access to specific hours (e.g., 9 AM to 5 PM). Physical Security • Cameras: Provide surveillance and deterrence. • Locks: Prevent unauthorized physical access to equipment rooms and racks. • Geofencing: Creating a virtual geographic boundary that triggers an action when a device enters or exits.
