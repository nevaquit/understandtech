Metric Definition Impact Bandwidth Maximum data transfer capacity of a link

Determines theoretical maximum throughput

Throughput Actual data transfer rate achieved Affected by congestion, errors, protocol overhead Latency Time for a packet to travel from source to destination

High latency causes slow response times

Jitter Variation in latency Disrupts real-time applications (VoIP, video) Packet Loss Percentage of packets that fail to reach destination

Causes retransmissions; degrades performance

Wireless Performance Issues

Issue Cause Solution

Channel Overlap Adjacent APs using overlapping channels Use non-overlapping channels (1, 6, 11 for 2.4 GHz)

Interference Microwaves, cordless phones, neighboring Wi-Fi

Switch to 5 GHz; use Wi-Fi analyzer to identify interference Insufficient Coverage AP placement or power too low Add APs; increase transmit power; adjust antenna placement Roaming Issues Client not roaming between APs smoothly Implement 802.11r (fast BSS transition); adjust AP power levels
