Software Tools

Tool Function Key Use Case ping Tests reachability using ICMP echo "Is the host alive?" traceroute/tracert Traces the path to a destination "Where is the connection failing?" nslookup Queries DNS servers "Is DNS resolving correctly?" dig Advanced DNS query tool (Linux/Mac) Detailed DNS troubleshooting netstat Displays active connections and listening ports "What ports is this machine listening on?" ip/ifconfig/ipconfig Displays/configures network interface settings "What is my IP address and gateway?" arp Displays/modifies ARP cache "What MAC address is associated with this IP?" tcpdump Command-line packet capture Capture traffic for analysis Wireshark GUI packet analyzer Deep packet inspection Nmap Network scanner and port scanner Discover hosts and open ports

LLDP/CDP Link Layer / Cisco Discovery

Protocol

Protocol Analyzer Captures and decodes packets at multiple OSI layers

Deep inspection of traffic flows

Speed Tester Measures actual throughput of a connection

Verify ISP bandwidth; identify bottlenecks

IP Scanner Discovers all active hosts on a subnet

Network inventory and rogue device detection

Hardware Tools

Tool Function Toner (Tone Generator and Probe) Traces a specific cable through a bundle or wall Cable Tester Verifies continuity, pinout, and wire map of a cable Wi-Fi Analyzer Measures signal strength, identifies channel overlap Taps (Test Access Points) Passive devices that copy network traffic for monitoring Visual Fault Locator Sends visible light through fiber to locate breaks or bends OTDR Measures fiber length and locates faults

Basic Networking Device Commands

Command Function show mac-address-table Displays the CAM table (MAC to port mappings) show route Displays the routing table show interface Displays interface status, errors, and statistics show config Displays the running configuration show arp Displays the ARP table show vlan Displays VLAN assignments show power Displays PoE status and power budget

Exam focus

Given a scenario, use the appropriate tool or protocol to solve networking issues

  • ping/traceroute for reachability; nslookup/dig for DNS.
  • tcpdump/Wireshark for packet-level evidence.
  • Cable certifiers and toners for physical layer.
  • show route / show interface on network OS for local state.
