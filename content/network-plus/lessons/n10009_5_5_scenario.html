Given a scenario, use the appropriate tool or protocol to solve networking issues

During a change window at Summit Logistics, the field technician must resolve a production networking issue tied to objective 5.5. Document findings using the troubleshooting methodology before escalating to Tier 3.

Situation briefing

  • Organization: Summit Logistics (warehouse networking)
  • Your role: field technician
  • Constraint: Maintenance window ends in 4 hours; rollback plan required
  • Evidence available: Packet captures, switch configs, monitoring dashboards (synthetic)

Concepts to apply (from objective 5.5)

  • Software Tools
  • LLDP/CDP Link Layer / Cisco Discovery
  • Protocol
  • Identify directly connected neighbors
  • Deep inspection of traffic flows
  • Verify ISP bandwidth; identify bottlenecks

Deliverables for this scenario

  1. Identify the most likely root cause using layered troubleshooting (physical → data link → network).
  2. State which tool or show command would confirm your hypothesis.
  3. Propose two remediation steps with rollback criteria.
