Given a scenario, troubleshoot common cabling and physical interface issues

Common Cable Problems

Issue Cause Solution

Attenuation Signal loss over distance Use appropriate cable length; add repeater/switch

Crosstalk Interference between adjacent wire pairs

Use properly shielded cable; maintain bend radius

EMI/RFI External electromagnetic interference

Use shielded cable (STP); route away from interference sources

Issue Cause Solution

Improper Termination Incorrect pinout or poor crimp Re-terminate; verify T568A or T568B standard TX/RX Transposed Transmit and receive wires swapped Use crossover cable or verify auto-MDIX is enabled

Interface Counter Errors

Counter Meaning Likely Cause

Runts Frames smaller than 64 bytes Collision; duplex mismatch Giants Frames larger than 1518 bytes Jumbo frames not configured; MTU mismatch CRC Errors Cyclic Redundancy Check failures Physical layer problem; bad cable or NIC Drops Frames dropped due to buffer overflow

Congestion; speed mismatch

Port Status

  • Administratively Down: Disabled by an administrator (shutdown command). Must be manually re-enabled.
  • Error-Disabled: Automatically shut down by the switch due to a violation (e.g., port security, BPDU guard).

Requires manual intervention to re-enable.
